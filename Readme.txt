这是一个做到一半的合作数独程序，基本框架已经搭建好了，还差一点游戏逻辑。
这个游戏的想法是大家一起联网解数独题。特点是支持局域网乱入联网合作解题。也就是说只要局域网内几台电脑同时开了这个程序，便可以自动进行匹配，合作解题。不过目前游戏逻辑模块还没有做完，启动后可以看到背后的命令行窗口里联网乱入后互相发送的一些测试数据。
做这个程序的起因是想跟老婆一起玩数独题，以前msn上有联网合作解题的，可惜现在没了，也没找到合适的程序，于是想自己做一个。不过，没多久便对数独不怎么感兴趣了，所以耽搁了下来。

CompressDataBase	压缩题库关卡的模块，已完成
DownloadDataBase	下载题库关卡的模块，已完成
Game			主游戏模块，未完成
Launcher		游戏启动模块，已完成
Network			网络模块，已完成
Puzzle			关卡模块，未完成
Render			渲染模块，未完成
SimpleCompression	简单关卡压缩模块，未完成
